## What is this?

A comprehensive, single-page interactive website that teaches you everything about **RAG (Retrieval-Augmented Generation)** — the technique powering modern AI applications that need real-world knowledge.

No frameworks. No build steps. Just pure HTML, CSS & JavaScript crafted for an immersive learning experience.

---

## Features

🤖 **Animated Robot Mascot** — A friendly guide that floats alongside your learning journey

🔬 **Interactive RAG Pipeline** — Click through all 8 steps (Ingest → Generate) with detailed explanations and tags

✂️ **Live Chunking Demo** — See how 4 different chunking strategies (Fixed, Sentence, Semantic, Recursive) split text in real-time

🌌 **Vector Space Visualization** — 2D canvas showing how embeddings cluster similar concepts together

📊 **6 Vector Database Cards** — Pinecone, Weaviate, ChromaDB, Milvus, Qdrant, pgvector compared with features & scale

🧩 **RAG Architecture Patterns** — Tabbed exploration of Naive, Advanced, Modular & Agentic RAG with pros/cons

🗺️ **Learning Roadmap** — 6-stage accordion timeline from Foundations to Cutting Edge with resources

🧪 **10-Question Quiz** — Test your RAG knowledge with instant feedback, explanations & scoring

✨ **Particle Background** — Canvas-based particle system with connecting lines

🎨 **Scroll Animations** — Intersection Observer-powered reveal animations throughout
